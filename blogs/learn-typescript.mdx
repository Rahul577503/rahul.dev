## Install

- **Note:** You need to have npm installed.
- Avoid using TypeScript globally in a team to prevent version conflicts.

```bash
npm install -g typescript
```
## File Extensions

- TypeScript files have extensions `.ts` or `.tsx` (for React JSX files).

## Compile

To compile TypeScript files:

```bash
tsc --watch
```

## Run

To run TypeScript code:

```bash
node index.js
```

**Tip:** Use `nodemon` to watch for changes and restart the server automatically.

```bash
npm install -g nodemon
nodemon index.js
```

Or directly use `nodemon` with `.ts` files:

```bash
nodemon index.ts
```

## Config File

Use `tsconfig.json` to configure TypeScript. You can generate a config file with `tsc --init`.

```json
{
  "outDir": "./dist",
  "rootDir": "./src",
  "target": "es6"
}
```

## Pick and Omit

Pick and Omit are utility types in TypeScript:
<br/>

<CodeBlock code ={`

interface User {
  id: number;
  name: string;
  lastName: string;
  age: number;
}

type Username = Pick<User, 'name' | 'lastName'>;

type UserWithoutId = Omit<User, 'id'>;

`} language ="ts" filename="utilityTy.ts"/>
